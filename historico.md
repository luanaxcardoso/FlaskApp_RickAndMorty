.venv\Scripts\activate (no adm powershell)

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser (no adm powershell)

pip install flask

---------------------------------------------------------------------------
No terminal para rodar o servidor, digite o seguinte comando:

flask --app app run

ou

python app.py
